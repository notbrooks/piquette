# Welcome to Piquette

This is a project generated from [Theo's T3 App Router Tutorial on YouTube](https://www.youtube.com/watch?v=w-l-Z-h-yfI). There's a lot of cool stuff here that Theo has done (insanely well), but I've taken a different approach to the project.  For those who are interested, the name Piquette is a referecne to the Ford Motor Company's factory where the Model T was made.  The Model T was the first car to be mass-produced in the United States, and it was a revolutionary car that changed the face of the automotive industry.  The factory was located in Detroit Michigan on Piquette Avenue.  Hence the name.

The goal of this project is to showcase how to build a full-stack application with Next.js, Postgres, and Vercel and effectivey set up the factory.  Get it?  As a developer, architect and failed designer of nearly 30 years - this app is intented to be a cumination of all of the set-up that happens on literally every project I've ever worked on.  I've tried to make it as easy as possible to get started, but I'm sure there are still a lot of things that could be improved.  If you have any questions, feel free to reach out to me by email or on Twitter.

Piquette is a low-code application meaning that there's a lot that still needs to be developed in order to make it a fully functional application.  My goal here is to make all of the hard stuff simple and let the developer focus on the core functionality of the application.  It's not intened to be a full-featured application, but rather a starting point for anyone who wants to build a full-stack application with Next.js, Postgres, and Vercel.
